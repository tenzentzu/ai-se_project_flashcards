# Flashcard App

My first project in TripleTen's AI-Assisted Software
Engineering program. It includes decks of flashcards,
each of which can be viewed in a carousel.

## Features

Contains a gallery of decks containing ten flashcards each. When a deck is selected,
a carousel view comes up that provides questions and answers for user to flip through.

## Technologies Used

HTML, CSS, Javascript
